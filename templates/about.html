About This Project

Powered by Machine Learning & Flask

Project Objective

This web application helps predict the actual productivity of garment factory employees using various input parameters like overtime, idle time, number of workers, and more.

Technology Stack

  • Frontend: HTML5, CSS3 (W3.CSS)
  • Backend: Python, Flask
  • Deployment Ready: IBM Cloud (included)

Machine Learning Models Used

  • Random Forest Regressor: Used for final prediction due to best performance.
  • Linear Regression: Provided a simple, interpretable baseline.
  • XGBoost Regressor: Tried for gradient boosting-based performance optimization.

Features

  • Interactive form for predictions
  • Clean, responsive UI
  • Real-time machine learning output
